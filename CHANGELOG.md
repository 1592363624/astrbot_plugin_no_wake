# 更新日志

本项目的所有重要更改都将记录在此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本控制](https://semver.org/lang/zh-CN/)。

## [1.0.1] - 2026-06-26

### 新增
- **自动补丁功能**：插件加载时自动检测并修补 AstrBot 核心的 `stage.py` 文件，确保 `"*"` 唤醒词逻辑在 AstrBot 更新后仍然生效
- **管理员通知**：补丁执行成功或失败时，通过QQ私聊主动通知管理员
- **防重复修补机制**：通过补丁标记 `[no_wake_patched]` 精确检测已修补文件，避免重复修补
- **动态路径查找**：自动查找 `stage.py` 文件路径，支持不同的安装结构
- **配置项扩展**：
  - `auto_patch`：启用/禁用自动补丁功能（默认开启）
  - `notify_admin`：启用/禁用管理员通知功能（默认开启）

### 优化
- **状态显示增强**：`免唤醒 status` 命令现在显示核心补丁状态
- **日志记录**：补丁操作结果会记录到 AstrBot 日志系统

### 修复
- 修复配置访问方式：使用 `context.get_config()` 替代不存在的 `context.astrbot_config`
- 修复路径计算逻辑：支持 AstrBotLauncher 等不同安装结构

## [1.0.0] - 2026-06-25

### 新增
- **免唤醒词功能**：允许机器人不需要唤醒词即可触发指令
- **动态开关控制**：
  - `免唤醒 status` - 查看当前状态
  - `免唤醒 on` - 开启功能
  - `免唤醒 off` - 关闭功能
  - `免唤醒 private` - 仅私聊启用
  - `免唤醒 group` - 仅群聊启用
  - `免唤醒 all` - 所有场景启用
- **唤醒词配置管理**：通过修改 AstrBot 主配置的 `wake_prefix` 实现免唤醒
- **多别名支持**：指令支持 `免唤醒`、`nowake`、`nowakeword` 别名

## 版本说明

### 版本号规则
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型
- **新增**：新功能
- **变更**：对现有功能的变更
- **弃用**：不建议使用，将在未来移除的功能
- **移除**：已移除的功能
- **修复**：任何 bug 修复
- **安全性**：安全相关的更改
